# NanoRpcSharp

C# (`netstandard2.0`) client library for [NANO (ex RaiBlock)](https://nano.org/en) RPC servers.

Based on [NANO RPC Wiki on GitHub](https://github.com/nanocurrency/raiblocks/wiki/RPC-protocol).
